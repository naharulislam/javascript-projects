# 🎓 Student Grade Management System

A simple command-line **Student Grade Management System** built with **JavaScript (Node.js)**. This project allows users to search for a student by roll number and view their academic information through an interactive menu.

---

## 📌 Features

- 🔍 Find a student using their roll number
- 👤 Display student details
- 📊 Calculate total marks
- 📈 Calculate average marks
- 🏆 Find the highest mark
- 📉 Find the lowest mark
- 🎯 Calculate student grade
- 📋 Interactive command-line menu
- ❌ Handles invalid roll numbers and menu options

---

## 🛠️ Technologies Used

- JavaScript (ES6)
- Node.js
- Readline Module

---

## 📖 How It Works

1. Enter a student's roll number.
2. If the student exists, an interactive menu will appear.
3. Choose one of the following options:

```text
1. Show Student Details
2. Show Total Marks
3. Show Average Marks
4. Show Grade
5. Find Highest Mark
6. Find Lowest Mark
7. Exit
```

4. The selected result will be displayed.
5. Continue using the menu until you choose **Exit**.
